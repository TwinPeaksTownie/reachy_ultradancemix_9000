# Purpose

Mainly a list of scripts of various levels of robustness...

# Usage
Make sure these 2 repos are installed in the same Python environment:
https://github.com/pollen-robotics/reachy_mini
https://github.com/pollen-robotics/reachy_mini_dances_library

For example:
```
git clone git@github.com:pollen-robotics/reachy_mini.git
git clone git@github.com:pollen-robotics/reachy_mini_dances_library.git

cd reachy_mini
pip install -e .


cd ../reachy_mini_dances_library
pip install -e .
```
